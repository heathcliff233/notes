# Chapter 02 随机变量及其分布

## Section 4 随机变量函数的分布

### 二、连续型随机变量函数的分布

定理：设随机变量 $X$ 的概率密度为 $f_X(x)$ ，$-\infin < x < \infin$  ，又设函数 $g(x)$ 处处可导且有 $g'(x)>0 \ or \ g'(x)<0$ ，则 $Y=g(X)$ 的概率密度为：
$$
f_Y(y) = \left\{ 
\begin{array}{rc1}
f_X(h(y))\cdot|h'(y)|, &\alpha<y<\beta  \\
0, &else
\end{array}
\right.
$$
其中 $\alpha=\min(g(-\infin),g(\infin))$ ， $\beta=\max(g(-\infin),g(\infin))$ ， $h(y)$ 是 $g(x)$ 的反函数
